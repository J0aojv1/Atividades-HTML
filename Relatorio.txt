Foi feito o processo para mandar um projeto do computador para o GitHub. Primeiro o git foi configurado e aberta a pasta do projeto. Depois foi iniciado o repositório, adicionados os arquivos e feito o primeiro commit. Em seguida foi criado o repositório no GitHub e ligado com o comando remote add origin. Por fim os arquivos foram enviados com o comando push. Também foram vistos os passos para atualizar o projeto quando tiver mudanças.
